### 快速开始
克隆项目文件:
```bash
https://github.com/chenfengyanyu/my-antd-demo2.git
```

进入目录安装依赖:
```bash
npm i
```

启动项目，打开 http://localhost:8000预览：
```bash
npm run start
```

构建生成dist目录：
```bash
npm run build
```

语法测试：
```bash
npm run test
```
